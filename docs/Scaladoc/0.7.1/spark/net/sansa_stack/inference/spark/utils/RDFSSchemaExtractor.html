RDFSSchemaExtractor

Related Doc: package utils

class RDFSSchemaExtractor extends Logging with Serializable

An extractor of the schema for RDFS.

Currently, it's supports the extraction of triples (s,p,o) with p being

  • rdfs:subClassOf
  • rdfs:subPropertyOf
  • rdfs:domain
  • rdfs:range
